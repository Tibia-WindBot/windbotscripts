[CENTER][SIZE=7][B]Safe Reconnect[/B][/SIZE][/CENTER]


[SIZE=4][B]Description[/B][/SIZE]

A safe reconnecter that will logout and disable itself if you are inside a temple or got skulled. Enable this hotkey after you logged in.

[SIZE=4][B]Configuration[/B][/SIZE]

Set [I]ReopenBps[/I] to [COLOR="#0033ff"][B]true[/B][/COLOR] if you want to reopen backpacks after login, it'll open all visible backpacks on the main bp. Set if to [COLOR="#0033ff"][B]false[/B][/COLOR] to ignore this option.
Set [I]IgnoreSS[/I] to [COLOR="#0033ff"][B]true[/B][/COLOR] to ignore the server save time, set to [COLOR="#0033ff"][B]false[/B][/COLOR] to wait the server save time.
